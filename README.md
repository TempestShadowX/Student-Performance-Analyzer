# Student Performance Analyzer

A Python script that reads student scores from a CSV file and analyzes their performance.

---

## What It Does

- Calculates each student's average score across all subjects  
- Shows who passed and who failed  
- Finds the highest and lowest performer  
- Displays overall class average and pass rate  
- Optional bar chart and pie chart visualization
